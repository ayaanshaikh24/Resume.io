# resumeio

A client-side resume and portfolio builder with real-time preview, print-perfect CSS, localStorage autosave, and a rule-based ATS resume score checker.

Built with [Astro](https://astro.build), React, TypeScript, and Tailwind CSS v4.

## Quick start

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
npm run preview
```
